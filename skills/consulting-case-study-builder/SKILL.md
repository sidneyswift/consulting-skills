# Consulting Case Study Builder

## Steps
1. Gather delivery docs and outcomes from `clients/{client}/04-delivery/` and any value reviews.
2. Structure: problem → approach → quantified outcome.
3. Save to `proof/case-studies/`.
4. Feed it back into the flywheel: queue a content piece and use it as a 24h value-add asset in sales.
5. Confirm any sensitive details are cleared for external use.
