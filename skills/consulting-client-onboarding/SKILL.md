# Consulting Client Onboarding

## Steps
1. Copy `clients/_TEMPLATE/` → `clients/{client}/` (full lifecycle subfolders).
2. Fill the dashboard `README.md`: status, engagement/value, champion, stakeholders, stakes, dates, next action.
3. Move the deal folder from `pipeline/` into the new client folder (artifacts: discovery → `01-discovery/`, proposal → `02-proposals/`).
4. Generate the kickoff checklist: SOW signed, first invoice, Slack Connect, repo provisioned, designated contacts, 48h-decision SLA.
5. Update `pipeline/_board.md`, the CRM stage (Closed Won), and `operating/dashboard.html`.
6. Propose the maintenance/retainer tier before delivery even starts.

Source: Ch. 16.
