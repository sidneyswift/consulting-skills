# Consulting Discovery Analysis

## When to run
After a discovery call or when a new lead transcript needs triage.

## Steps
1. Read the transcript.
2. Produce:
   - What is truly at stake (specific business/$ impact)
   - Top 3 pain points in order of urgency
   - Success metrics mentioned or implied
   - Objections / concerns raised
   - Questions that should have been asked but weren't
   - Qualified buyer? Why / why not (budget authority, stakes, timeline)
3. Save the analysis to the deal/client's `01-discovery/` folder as `discovery-analysis-YYYY-MM-DD.md`.
4. If qualified, chain into `consulting-proposal-drafting`.
