# Consulting Quarterly Value Review

## Steps
1. Pull the client's delivery log and notes from `clients/{client}/04-delivery/`.
2. Assemble: problems solved · value created · time saved · opportunities opened (quantify where possible).
3. Write the review to `clients/{client}/05-retention/value-review-YYYY-MM-DD.md`.
4. Tee up next steps: a renewal/expansion proposal (→ `consulting-expansion-spotter`) and a testimonial ask (→ `consulting-testimonial-capture`).
5. Update the client dashboard README.

Source: Ch. 17.
