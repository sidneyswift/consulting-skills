# Consulting Expansion Spotter

## Steps
1. Scan the client's notes, delivery, and value review for adjacent problems and new stakeholders.
2. Map Land → Expand → Transform moves and a possible tier upgrade.
3. Draft an expansion proposal (new SOW under the existing MSA) to `clients/{client}/06-expansion/`.
4. Time the ask to a win or a value review.
