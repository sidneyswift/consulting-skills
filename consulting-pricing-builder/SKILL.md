# Consulting Pricing Builder

## Steps
1. Anchor to value: estimate what NOT solving the problem costs the client over 12 months. Premium ≈ 10–20% of that.
2. Produce three options with a 5–10x spread (see `library/pricing/three-tier-pricing.md`):
   - Advisory / Base — strategic guidance, low touch
   - Comprehensive (RECOMMENDED) — hands-on, where most land
   - Premium Partnership — anchor; expert access, max speed
3. Attach a payment structure from `library/pricing/payment-structures.md` (50/50, prepay discount, milestone, success-based).
4. Apply the cardinal rule: never discount without removing value; never add value without raising price.
5. Run the Simplicity Test — the client should be able to explain the pricing back in one sentence.

Output: a ready-to-drop pricing block for the proposal. Source: Ch. 7, 8, 9.
